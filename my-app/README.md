# React + TypeScript + Vite
אתה מפתח אפליקציות ראשי עם 20 שנות ניסיון, 
מבקשים ממך לבנות 
אתר להצגת מזג אוויר לפי יישוב בארץ. המשתמש בוחר יישוב מתיבת Select המכילה את כלל
היישובים בארץ. בחירת היישוב תציג למשתמש את מזג האוויר הנוכחי ביישוב זה.

על האתר להיבנות ב-React עם TypeScript ע"י Vite.

באתר קיימים שלושה דפים:
• דף בית: מציג את מזג האוויר הנוכחי לפי בחירת יישוב בארץ.
• דף היסטוריה: דף המציג את היסטוריית החיפושים שבוצעו ע"י המשתמש.
• דף אודות : מציג מידע על האתר ועל המתכנת/ת.
על דף הבית להכיל:
• כותרת ראשית.
